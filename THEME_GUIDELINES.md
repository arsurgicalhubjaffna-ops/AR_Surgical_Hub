# THEME GUIDELINES - AR Surgical Hub

This document defines the visual language, color palette, and design principles for the AR Surgical Hub web application.

## Design Philosophy
The design should feel **Premium**, **Clean**, and **Trustworthy**. Use a mix of deep blues, medical teals, and professional whites.

## Color Palette

### Primary Colors
- **Deep Navy**: `#0A192F` (Backgrounds, Headers)
- **Medical Teal**: `#14B8A6` (Primary Buttons, Accents)
- **Soft Cyan**: `#22D3EE` (Links, Hover states)

### Secondary Colors
- **Cloud White**: `#F8FAFC` (Surface backgrounds)
- **Steel Gray**: `#64748B` (Subtext, Borders)
- **Error Red**: `#EF4444` (Alerts, Destructive actions)

### Dark Mode (Preferred)
- **Dark Bg**: `#0F172A`
- **Surface**: `#1E293B`
- **Border**: `#334155`

## Typography
- **Primary Font**: `Inter`, sans-serif (Clean, modern)
- **Header Font**: `Outfit`, sans-serif (Geometric, professional)

## Component Styles

### Buttons
- **Primary**: Gradient of Medical Teal to Soft Cyan. Rounded-lg, shadow-lg on hover.
- **Secondary**: Ghost style with Steel Gray border.

### Cards
- Glassmorphism effect for dark mode surfaces.
- Border radius: `12px`.
- Subtle shadow: `0 10px 15px -3px rgba(0, 0, 0, 0.1)`.

## Micro-Animations
- Use smooth transitions (`200ms ease-in-out`) for hover states.
- Subtle fade-in for page entries.
